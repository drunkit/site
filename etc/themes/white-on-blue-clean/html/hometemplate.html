Welcome to the all new drunkit.co.uk home page. Over the next few week, expect massive changes, including having a unified account (i.e. your forum username/password will be the same as your Drunkit one).

This is a new template that I'm playing with...

Notice how the menu can collapse. This works with Mozilla, Internet Explorer, Opera, Safari, Konqueror and probably others too. It detects if this isn't supported and hides the option if it's not possible.

Quite a clean interface as well as a fairly small HTML footprint. A lot of CSS positioning - only one table is used (although it's not used for document structure, only to enable easy alignment of the right arrow when the menu is hidden.
